Install
------------------------------------------------------------------------------

``learn_aws_code_artifact`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_aws_code_artifact

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_aws_code_artifact
